# Use of icons

##==##

<!-- .slide: class="with-code" -->

# Configurables icons

You can use icons using the following syntax:

```markdown
![](icon-keyword 'icon-framework-keyword tc-icons')
```

<!-- .element: class="big-code" -->

Supports libraries:

-   Font Awesome
-   Feather
-   Material Design Icons
